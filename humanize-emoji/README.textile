h1. Unified Emoji for Java

h2. Features

* Search by code points, raw characters and annotations
* Unified Emoji character database
* UTF-16 encoding support
* Vendor codes mapping (DoCoMo, KDDI, SoftBank)
* Sexy and easy text interpolation of emoji code points and aliases.
