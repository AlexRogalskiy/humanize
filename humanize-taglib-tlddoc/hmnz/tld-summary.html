Humanize Tag Library

Standard Syntax:
     <%@ taglib prefix="hmnz" uri="http://mfornos.github.com/humanize/taglib" %>
XML Syntax:
     <anyxmlelement xmlns:hmnz="http://mfornos.github.com/humanize/taglib" />
 
Humanize formatting library

Tag Library Information
Display NameHumanize Tag Library
Version1.1
Short Namehmnz
URIhttp://mfornos.github.com/humanize/taglib
 

Tag Summary
message Maps key to localized message and performs parametric replacement
param Supplies an argument for parametric replacement to a containing tag
pluralize Constructs a message with pluralization logic from the given template.
naturalTime Computes both past and future relative dates. E.g. 'one day ago', 'one day from now', '10 years ago', '3 minutes from now', 'right now' and so on.
naturalDay String with 'today', 'tomorrow' or 'yesterday' compared to current day. Otherwise, returns a string formatted according to a locale sensitive DateFormat.
slugify Transforms a text into a representation suitable to be used in an URL.
binaryPrefix Converts a given number to a string preceded by the corresponding binary International System of Units (SI) prefix.
metricPrefix Converts a given number to a string preceded by the corresponding decimal multiplicative prefix.
ordinal Converts a number to its ordinal as a string.
nanoTime Formats a number of nanoseconds as the proper ten power unit.
spellBigNumber Converts a big number to a friendly text representation. Accepts values ranging from thousands to googols. Uses BigDecimal.
spellDigit For decimal digits [0-9], returns the number spelled out. Otherwise, returns the number as string.
